# EMBA 206 - Marketing Management Study Guide

Interactive study app for **EMBA 206: Marketing Organization and Management** at UC Berkeley Haas (Spring 2026, Prof. Ganesh Iyer).

Built with React + Vite + Tailwind CSS. Deployed on GitHub Pages.

**Live site:** https://rohitmishrajan18.github.io/Marketing_management/

## What's Inside

The app is structured as a four-phase learning journey:

### Learn
12 concept lessons organized by course block, each with explanations and key takeaways.

- **Block 1 -- Customer Value:** Marketing concept, CLV, Two Sides of Value, STP
- **Block 2 -- Product & Price:** Product strategy, conjoint analysis, pricing, product line pricing
- **Block 3 -- Communication:** Advertising strategy, promotions, distribution channels
- **Block 4 -- Analytics & AI:** Promotion evaluation, marketing analytics

### Review
Quick-reference cheat sheets for exam day.

- All formulas (CLV, conjoint, EVC, product line, promotions)
- 8 strategic frameworks with when-to-use guidance
- 20 key term comparisons (vertical vs horizontal, pull vs push, etc.)

### Practice
Step-by-step solution templates with worked examples from class.

- 5 problem types: CLV calculation, conjoint analysis, product line pricing, EVC pricing, promotion evaluation
- 6 case study breakdowns: Chase Sapphire, Olay, On Shoes, Coke Vending Machines, TikTok, Formlabs

### Test
Self-assessment tools to check your readiness.

- 35 flashcards with flip animation
- 15-question multiple choice quiz with instant feedback and scoring
- 20 interactive exam traps (common misconceptions to watch for)

## Exam Details

Closed book, 2 pages (8.5"x11") of self-made help notes allowed, 2.5 hours, 100 points.

## Local Development

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` trigger automatic deployment via GitHub Actions. To set up:

1. Go to **Settings > Pages > Source** and select **GitHub Actions**
2. Push to `main` -- the workflow handles the rest
